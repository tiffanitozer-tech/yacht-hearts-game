# Yacht Hearts Visual Prototype

This is the next visual prototype for the original interactive dating game.

## What is included
- Colorful app-style UI
- Customizable player avatar
- Outfit closet with visual previews
- Yacht scene layout
- Character/cast cards
- Relationship meters
- In-game phone messages
- Episode 1 scene prototype

## How to upload to GitHub
1. Unzip this folder.
2. Open your GitHub repository.
3. Upload these three files to the root of the repository:
   - index.html
   - style.css
   - game.js
4. Commit changes.
5. Your GitHub Pages link will update automatically after a minute or two.

## Important
This uses CSS-based placeholder artwork. That means it loads fast and works on GitHub Pages without needing extra image files.
Later, these placeholder characters can be replaced with custom illustrated PNG/SVG artwork.
